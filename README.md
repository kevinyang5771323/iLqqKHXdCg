# Python Web餐厅后台管理系统 java1087

## 项目概述

本项目是基于Python语言的Web餐厅后台管理系统，使用Django框架进行开发，数据库采用Oracle。系统主要实现了登录注册、员工信息管理、菜品管理、收银员信息管理、点餐界面及订单管理等核心功能。

## 技术栈

- **后端：** Python
- **框架：** Django
- **数据库：** Oracle

## 功能模块

- **用户模块：**
  - 登录
  - 注册
- **信息管理模块：**
  - 收银员信息管理
  - 员工信息管理
- **菜品管理模块：**
  - 菜品添加
  - 菜品编辑
  - 菜品删除
- **点餐模块：**
  - 大堂点餐界面
- **订单管理模块：**
  - 查看订单
  - 订单管理

## 系统角色

- 系统管理员
- 收银员
- 厨师
- 服务员
